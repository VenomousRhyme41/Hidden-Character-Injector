## What You Can Do

**Inject Hidden Characters**
Add invisible Unicode characters (U+200B, U+200C, U+200D) between words. Useful for watermarking text, creating unique strings, or testing hidden content detection.

**Full Control**
Set exactly how many characters go between each word, how often they appear (every word, every other word, etc.), and which hidden characters to use, choose one or multiple.

**Reveal Mode**
Toggle "Reveal Hidden" to make invisible characters visible as highlighted ␣ symbols, so you can see exactly where they're placed without losing the ability to copy the raw text.

**Max Limit Protection**
Set a maximum total of hidden characters (up to 15,000) to prevent over injection. The "MAX" button instantly sets it to the limit, perfect for filling text to a specific length.

## Quick Start

1. Go to the [live site](https://venomousrhyme41.github.io/Hidden-Character-Injector/)
2. Paste or type your text into the source box
3. Choose your settings:
   - Pick a preset (Light, Medium, Extreme, Random)
   - Select which hidden characters to use
   - Set how many per gap and how often
   - Adjust the max total if needed
4. Copy the output with invisible characters
5. Click "Reveal Hidden" to see where they are

That's it. No setup, no downloads, no accounts needed.

## Settings Explained

| Setting | What It Does |
|---------|-------------|
| **Presets** | Quick setups: Light (1/10), Medium (3/3 mixed), Extreme (7/1 all chars), Random (everything randomized) |
| **Hidden Chars** | Choose from ZWSP (U+200B), ZWNJ (U+200C), ZWJ (U+200D)—multi-select allowed |
| **Per Gap** | Number of hidden characters inserted each time |
| **Gap Every** | How often to insert: 1 = every word, 2 = every other word, etc. |
| **Max Hidden** | Hard limit on total hidden characters injected |
